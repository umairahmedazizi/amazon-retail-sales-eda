# Data

The full datasets are not kept in this repository. They are large, and GitHub rejects any single
file over 100 MB.

| File | Size | In the repo? |
|------|------|--------------|
| `amazon_sales_dataset.xlsx` (raw) | about 49 MB | No |
| `amazon_sales_cleaned.xlsx` (cleaned) | about 24 MB | No |
| `amazon_sales_cleaned_sample.csv` | small | Yes, a 1,000-row sample |

## Getting the full data

The original dataset is published on Figshare under the name "amazon_sales_data". Download it and
place the files in this folder, then run the notebook from start to finish. The cleaned file is
produced by the cleaning steps in the notebook, so you only really need the raw file to reproduce
everything.
